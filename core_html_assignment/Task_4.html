wwe are going to write some html tags

  1. DOCTYPE TAG
    DOCTYPE TAG is used in the top of the document. it tells browser about the document
  2. html TAG
    This is the main tag of the html document which contains all tags except " DOCTYPE tag "
  3. head TAG
    This is the head tag which is used for meta information(information about webpage).
  4. body TAG
    when we write any content in body tag , it is visible for users.
  5. div TAG
    if we want to group many tags we can write those tags in division tag.
  6. i tag
    It converts the text to the italic text
  7. b tag
    It converts the text to the Bold text
